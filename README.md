# Project Phoenix

## Professional Algorithmic Trading System

Project Phoenix is a modular, scalable, and professional algorithmic trading system developed using Python.

The project is designed according to enterprise software engineering principles and follows the architecture defined in:

- SSD V1.0
- SSS V1.0
- TDD V1.0
- PPP V1.0
- SMS V1.0
- PDA V1.0
